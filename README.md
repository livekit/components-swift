# Swift Components

<!--BEGIN_DESCRIPTION-->
Use this SDK to add realtime video, audio and data features to your SwiftUI app. By connecting to <a href="https://livekit.io/">LiveKit</a> Cloud or a self-hosted server, you can quickly build applications such as multi-modal AI, live streaming, or video calls with just a few lines of code.
<!--END_DESCRIPTION-->

## Docs

Docs and guides are at [https://docs.livekit.io](https://docs.livekit.io)

## Example App

See our [Meet Example app](https://github.com/livekit-examples/swift-components) for a simple conferencing app.
